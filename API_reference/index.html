    Справочник API

    Введение

    Примечание

    Чтобы скачать спецификацию API, перейдите по ссылке Спецификация API

    API Order Base — справочник запросов для Order Base, в котором описаны доступные методы и способы их применения.

    Методы API разделены на несколько категорий:

    • order — заказы.
    • items — изделия.

    Часто используемые методы

    На практике наиболее часто используемыми методами являются:

    1. Взаимодействовать с определенным заказом 
      /orders/{orderId}
    2. Взаимодействовать с изделиями 
      /items

    Пример запроса

    Чтобы выполнить операцию с определенным заказом, необходимо указать его id.

    Пример запроса получения информации о заказе будет выглядеть следующим образом:

    GET /orders/{orderId}
    где {orderId} — id заказа.

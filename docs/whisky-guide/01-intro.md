# Что такое Whisky?

**Whisky** – это бесплатная утилита с открытым исходным кодом, которая предоставляет простой графический интерфейс для запуска Windows-приложений на компьютерах Mac с процессорами Apple Silicon (M1, M2, M3 и новее).

## Как это работает?

Whisky не эмулирует компьютер, а выступает «переводчиком» между Windows-программами и macOS. Он использует два ключевых компонента:

- **Wine** (через наработки CrossOver) – преобразует системные вызовы Windows в вызовы, понятные Unix-системам.
- **Apple Game Porting Toolkit** – переводит графические API DirectX в Metal, что обеспечивает высокую производительность в играх.

Благодаря этому приложения работают быстро и без лишних накладных расходов, как если бы они были родными для macOS.

## Ключевое понятие: «Бутылка» (Bottle)

В Whisky все приложения запускаются внутри так называемых **бутылок**. Это изолированные виртуальные окружения, каждое из которых имитирует отдельную установку Windows со своим диском C, реестром и настройками.

- Для каждой игры или программы рекомендуется создавать отдельную бутылку.
- Это позволяет избегать конфликтов между разными приложениями (например, разным версиям нужны разные библиотеки).
- В любой момент бутылку можно удалить, не затрагивая другие программы.

## Когда использовать Whisky?

Whisky идеально подходит для:

- Запуска старых и не очень старых Windows-игр на Mac.
- Использования небольших Windows-утилит, которым не хватает аналогов под macOS.
- Тестирования приложений в окружении Windows без установки виртуальной машины.

> **Примечание:** Так как проект официально заморожен, не все новинки будут работать. Перед установкой конкретной игры стоит проверить её статус в [вики-списке совместимости](https://github.com/IsaacMarovitz/Whisky/wiki/Game-Support).

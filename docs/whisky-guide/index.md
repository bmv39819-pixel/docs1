# С чего начать?

Если вы впервые сталкиваетесь с **Whisky**, рекомендуем следовать по порядку:

1. [Что такое Whisky 🥃](01-intro.md) — обзор и ключевые понятия («бутылки», Wine, Game Porting Toolkit).
2. [Системные требования](02-requirements.md) — убедитесь, что ваш Mac подходит.
3. [Установка и запуск Whisky](03-installation.md) — пошаговая инструкция по установке и запуску.
4. [Создание и настройка «бутылки»](04-create-bottle.md) — изолированное окружение для приложений.
5. [Установка и запуск Windows-приложений](05-run-apps.md) — как установить и запустить любое Windows-приложение.
6. [Установка и запуск Steam](06-steam.md) — отдельное руководство для установки, запуска Steam и игр из него.
7. [Решение проблем](07-troubleshooting.md) — частые ошибки и способы их устранения.
8. [Whisky для Git‑хуков (для разработчиков)](08-alternative-php.md) — если вы искали PHP-инструмент с таким же названием.

!!! warning "Важно"
    Проект Whisky больше не поддерживается активно, поэтому некоторые игры и программы могут работать нестабильно. Тем не менее, для множества приложений это отличное бесплатное решение.

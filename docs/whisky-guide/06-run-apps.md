# Запуск приложений и игр (общий метод)

После создания бутылки вы можете устанавливать в неё любые Windows-программы. Процесс аналогичен работе на реальном ПК: вы запускаете установщик `.exe` или `.msi`, проходите мастер установки, а затем запускаете приложение.

## Установка одиночной программы

1. В левой панели Whisky выберите нужную бутылку.
2. Нажмите кнопку **«Выполнить»** (или **«Run»**). Откроется окно выбора файла.
3. В Finder найдите и выберите исполняемый файл (например, `setup.exe` или портабельную программу `game.exe`).
4. Если это установщик, следуйте его инструкциям. Обычно нужно нажимать «Next», принимать лицензию и выбирать папку установки (по умолчанию будет диск C: внутри бутылки).
5. После завершения установки программа появится в списке **«Установленные программы»** внутри бутылки. Вы можете закрепить её там для быстрого запуска (значок звёздочки или контекстное меню).

Теперь для запуска программы достаточно выбрать бутылку и дважды кликнуть по иконке приложения в списке.

## Запуск портабельных приложений

Некоторые программы не требуют установки и работают прямо из папки. В таком случае:

1. Скопируйте папку с программой в любое место внутри бутылки. Удобнее всего поместить её в `C:\Program Files` или `C:\Games`.
2. Чтобы открыть диск C: бутылки в Finder, нажмите правой кнопкой мыши по бутылке в списке и выберите **«Открыть в Finder»** (или аналогичный пункт).
3. Затем в Whisky нажмите **«Выполнить»** и укажите путь к EXE-файлу внутри этой папки.

## Что делать, если программа требует библиотеки?

Если при запуске появляются ошибки о недостающих DLL (например, `MSVCR140.dll` или `D3DX9_43.dll`), скорее всего, нужно установить соответствующие компоненты через **Winetricks**:

1. Выберите бутылку.
2. Нажмите кнопку **«Winetricks»** (или в меню бутылки выберите «Запустить Winetricks»).
3. В открывшемся окне выберите **«Install a Windows DLL or component»**.
4. Отметьте нужные библиотеки (например, `vcrun2022` для Visual C++ Redistributable, `dotnet48` для .NET Framework, `corefonts` для базовых шрифтов).
5. Нажмите OK и дождитесь установки.

После этого попробуйте запустить программу снова.

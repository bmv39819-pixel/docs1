# Создание и настройка «бутылки»

Прежде чем устанавливать какую-либо программу, нужно создать для неё изолированное окружение — **бутылку**. Рекомендуется для каждого приложения или игры создавать отдельную **бутылку**.

## Пошаговая инструкция

1. В главном окне **Whisky** нажмите на кнопку «Создать бутылку» (или «+ Create Bottle»).
2. В появившемся окне заполните поля:
      - **Название бутылки** — например, `Steam`, `Heroes3` или `MyApp`.
      - **Версия Windows** — для современных приложений выберите **Windows 10/11**. Для игр более ранних версий может потребоваться Windows XP/7.
      - **Архитектура** — оставьте 64-bit, если приложение не требует 32-битную среду.
3. Нажмите «Создать».

Новая **бутылка** появится в боковом меню слева. При выборе **бутылки** в правой части окна отобразятся её свойства и список установленных программ.

## Дополнительные настройки бутылки

Для каждой **бутылки** можно изменять параметры конфигурации:

- **MSync/ESync** — режимы синхронизации потоков. Переключение между ними помогает решить проблемы с производительностью или звуком.
- **D3DMetal/DXVK** — переключатели графических бэкендов. D3DMetal (по умолчанию) использует Game Porting Toolkit для преобразования DirectX 12 в Metal. Для игр более ранних версий на DirectX 9–11 лучше включить DXVK.
- **Winetricks** — кнопка для установки дополнительных библиотек (например, `vcrun2022`, `dotnet48`, шрифты). Это может потребоваться для корректной работы некоторых приложений.

Чтобы открыть настройки **бутылки**, выберите её и нажмите на значок шестерёнки или пункт «Configure».

---

## Удаление бутылки

Если **бутылка** больше не нужна, её можно удалить целиком. Это очистит все связанные с ней файлы и реестр. Для этого выберите **бутылку** и нажмите «Удалить» (в контекстном меню или в настройках).

# Первый запуск

1. Запустите Whisky из папки «Программы». Система может предупредить, что приложение загружено из интернета – разрешите запуск (кнопка «Открыть»).

2. При первом запуске утилита запросит установку дополнительных компонентов:

- Rosetta 2 – необходима для работы некоторых x86-компонентов Wine. Подтвердите установку.

- Сама среда Whisky загрузит необходимые файлы Wine и Game Porting Toolkit.

3. Дождитесь окончания настройки. После этого вы увидите главное окно приложения.

Если вы видите главное окно Whisky с предложением создать бутылку, значит, установка прошла успешно. Можно переходить к следующему шагу – созданию окружения.

# Whisky для Git‑хуков (для разработчиков)

Если вы попали в это руководство в поисках инструмента для управления Git-хуками в PHP-проектах, то, вероятно, вам нужен другой проект с тем же названием — **«projektgopher/whisky»**. Это консольная утилита, которая помогает командам разработчиков использовать единые Git-хуки (например, для автоматического запуска линтеров, тестов или форматирования кода перед коммитом).

## Установка в PHP-проект

Утилита устанавливается через Composer как зависимость для разработки:

```bash
composer require --dev projektgopher/whisky  
```  

После установки выполните команду инициализации:

```bash
./vendor/bin/whisky install 
```   

Эта команда создаст в корне вашего проекта файл конфигурации `whisky.json` и настроит Git так, чтобы хуки из этого файла выполнялись автоматически.

---

## Пример конфигурации whisky.json

```json
{
    "hooks": {
        "pre-commit": [
            "composer run lint",
            "composer run test -- --group=unit"
        ],
        "pre-push": [
            "composer run test"
        ]
    }
}
```

Теперь перед каждым коммитом будут выполняться указанные команды. Если какая-то из них завершится ошибкой, коммит будет отменён.

---

## Отличие от macOS-утилиты Whisky

Не путайте эти два инструмента:

- **Whisky (macOS)** — графическая программа для запуска Windows-игр на Mac.
- **Whisky (PHP)** — консольный инструмент для автоматизации Git-хуков в проектах на PHP.

Убедитесь, что вы используете нужный пакет. Для получения дополнительной информации о PHP-версии посетите репозиторий на GitHub.

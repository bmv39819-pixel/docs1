# Установка и запуск Steam

**Steam** — это один из самых популярных способов запуска Windows-игр через **Whisky**. Для него рекомендуется создать отдельную бутылку.

## Создание бутылки для Steam

1. В **Whisky** нажмите «Создать бутылку».
2. Назовите её, например, `Steam`.
3. Выберите версию Windows (Windows 10/11).
4. Нажмите «Создать».

---

## Установка Steam

1. Скачайте установщик **Steam** для Windows с официального сайта:  
   [https://store.steampowered.com/about/](https://store.steampowered.com/about/) (кнопка «Загрузить Steam»).
2. В **Whisky** выберите созданную бутылку `Steam` и нажмите «Выполнить».
3. Укажите путь к скачанному файлу `SteamSetup.exe`.
4. Пройдите стандартную процедуру установки. 
!!! warning "Важно" 
    На последнем шаге снимите галочку «Запустить Steam», если она есть.
После завершения установки **Steam** появится в списке установленных программ. Для быстрого запуска закрепите его, нажав на звёздочку рядом с иконкой.

---

## Запуск Steam и установка игр

1. Выберите бутылку `Steam` и дважды кликните по иконке **Steam** в списке программ.
2. Войдите в свой аккаунт. При первом входе может потребоваться подтверждение по почте — это нормально.
3. Перейдите в библиотеку и устанавливайте любые Windows-игры, как на обычном ПК. Они будут загружаться в папку `C:\Program Files (x86)\Steam` внутри бутылки.

---

## Важные советы

- **Одна игра — одна бутылка?** Если вы планируете играть во множество игр через **Steam**, можно оставить всё в одной бутылке. Однако если какая-то конкретная игра работает нестабильно, попробуйте создать для неё отдельную бутылку и установить **Steam** заново только для неё. Это поможет избежать конфликтов библиотек.
- **Производительность:** Для некоторых игр может потребоваться переключение между MSync/ESync или D3DMetal/DXVK в настройках бутылки. Экспериментируйте.
- **Самопроизвольный запуск Steam:** Если **Steam** запускается фоново без вашего ведома, убедитесь, что вы полностью вышли из него (правый клик по иконке в трее → «Выйти»). Также можно перезапустить бутылку или Mac.

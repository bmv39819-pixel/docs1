# Системные требования

Чтобы **Whisky** работал корректно, ваш Mac должен соответствовать следующим требованиям:

- **Процессор:** Apple Silicon (M1, M2, M3, M4 и новее). 
- **Операционная система:** macOS Sonoma (версия 14.0 и новее). 
  Это обязательное условие, поскольку Game Porting Toolkit встроен именно в эту и последующие версии системы.
- **Свободное место на диске:** зависит от устанавливаемых приложений, для самой утилиты достаточно 500 МБ.
- **Оперативная память:** рекомендуется 8 ГБ и более, для современных игр — 16 ГБ и более.

!!! note "Примечание"  
    Intel-процессоры официально не поддерживаются, так как Whisky опирается на Apple Game Porting Toolkit, доступный только на ARM-архитектуре.  

## Проверка версии macOS

Чтобы узнать версию вашей системы, нажмите на логотип Apple в левом верхнем углу → **«Об этом Mac»**. В открывшемся окне будет указана версия macOS.

Если у вас более ранняя версия системы (например, macOS Ventura), **Whisky**, к сожалению, не запустится. В этом случае можно рассмотреть аналоги вроде CrossOver или Parallels Desktop.

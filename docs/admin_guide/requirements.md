**Аппаратные требования:**

- Минимальные требования:
    - Процессор: 2-ядерный, 2 ГГц;
    - Оперативная память: 4 ГБ;
    - Свободное место на диске: 20 ГБ;
    - Видеокарта: с поддержкой разрешения 1920x1080;
    - Сетевой адаптер: Ethernet 100 Мбит/с или Wi-Fi.

- Рекомендуемые требования:
    - Процессор: 4-ядерный, 3 ГГц и выше;
    - Оперативная память: 8 ГБ и более;
    - Свободное место на диске: 50 ГБ;
    - Видеокарта: с поддержкой разрешения 1920x1080;
    - Сетевой адаптер: Gigabit Ethernet или Wi-Fi 5 (802.11ac).

**Программные требования:**

- Операционная система:
    - Поддержка ОС: Windows 10/11;
- Дополнительное ПО:
    - Установленная среда выполнения .Net Framework 4.8;
    - Для модуля необходим установленный .Net Core 8;
- Серверное ПО:
    - Нет.
